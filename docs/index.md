## Host it yourself

### Requirements

- Docker runtime
- Docker compose

### Start Server

1. Setup docker-compose.yml:

```yml
version: "3.8"
networks:
  redis-net:
services:
  redis:
    image: redis
    container_name: cache
    expose:
      - 6379
    networks:
      - redis-net
  rki-api:
    image: marlon360/rki-covid-server:v2
    ports:
      - "8080:3000"
    depends_on:
      - redis
    environment:
      - REDISHOST=redis
    networks:
      - redis-net
```

2. Start Server:

`docker-compose up`

Now you can access the server at `http://localhost:8080`.
